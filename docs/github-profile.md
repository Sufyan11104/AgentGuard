## LinkedIn Post

I built AgentGuard, an open-source defensive evaluation toolkit for AI agents and LLM applications.

It includes a CLI, Next.js dashboard, safe synthetic test packs, mock safe/vulnerable targets, fake canary leakage detection, dry-run tool-call policy checks, scoring, report generation, Prisma/PostgreSQL persistence, and CI-backed tests.

The goal is practical: help developers evaluate owned AI agents against common failure modes like prompt injection, unsafe tool calls, hallucinated actions, and unsafe output handling before those workflows reach production.

Repo focus: TypeScript, AI security, agent evaluation, clean monorepo architecture, and defensive safety boundaries.
